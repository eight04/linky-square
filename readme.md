Linky Square
============
An userscript which allows you to drag a square to open/copy links.

Check the [demo](https://rawgit.com/eight04/Linky-Square/master/demo.html) page.

Install
-------
[Greasy Fork](https://greasyfork.org/scripts/18006-linky-square)
